생성형 AI(Generative AI)란?

생성형 AI는 새로운 텍스트, 이미지, 오디오, 영상 등을 자동으로 만들어내는 인공지능 기술입니다. 입력(prompt)에 따라 창의적이고 실용적인 결과물을 생성합니다.

간단한 원리

  • 대규모 데이터로 학습된 모델이 패턴을 학습합니다.
  • 확률 모델을 사용해 다음에 올 단어나 픽셀을 생성합니다.
  • 언어모델(LLM), 이미지 생성 모델(예: diffusion) 등 다양한 아키텍처가 사용됩니다.

활용 사례

  • 텍스트 생성: 기사, 요약, 이메일 초안, 코드 생성
  • 이미지 생성: 광고, 디자인 시안, 아트워크 제작
  • 음성/오디오: 음성 합성, 음악 작곡
  • 제품 개발: 시나리오 기반 테스트 데이터 생성

윤리와 안전

  • 허위 정보(착각, hallucination) 위험을 관리해야 합니다.
  • 편향(bias)과 저작권 문제를 고려해야 합니다.
  • 민감한 사용처에서는 검증, 인간 감독, 정책이 필요합니다.
